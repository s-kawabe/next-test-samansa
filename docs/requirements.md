# 映画情報サービス 要件定義

## Purpose

ユーザーが映画をカテゴリ別に閲覧し、興味のある作品の詳細情報やコメントを確認できる映画情報閲覧サービスを提供する。カテゴリ一覧・映画一覧・映画詳細という階層構造のナビゲーションを通じて、ユーザーが目的の映画に素早くたどり着けるようにする。

---

## Requirement: トップページ（映画カテゴリ一覧）

カテゴリごとにグループ化された映画サムネイル一覧をトップページに表示する。

**ユーザーストーリー**: 映画を探しているユーザーとして、カテゴリ別に映画を一覧したい。なぜなら興味のあるジャンルからすぐに映画を見つけられるからだ。

### Scenario: カテゴリと映画の一覧表示

- **WHEN** ユーザーがトップページにアクセスする
- **THEN** カテゴリ名とそのカテゴリに属する映画のサムネイル一覧を表示する
- **THEN** 各カテゴリにカテゴリ詳細ページへのリンクを表示する
- **THEN** 各映画サムネイルに映画詳細ページへのリンクを付与する

### Scenario: データ取得中

- **WHEN** データの取得が完了していない
- **THEN** ローディング状態をユーザーに示す

---

## Requirement: カテゴリ詳細ページ

指定カテゴリに属するすべての映画サムネイルをカテゴリ詳細ページに表示する。

**ユーザーストーリー**: 特定カテゴリに興味があるユーザーとして、そのカテゴリの映画を網羅的に閲覧したい。なぜなら気になる映画を見落とさずに探せるからだ。

### Scenario: カテゴリ別映画一覧の表示

- **WHEN** ユーザーがカテゴリ詳細ページにアクセスする
- **THEN** 対象カテゴリ名を表示する
- **THEN** そのカテゴリに属する映画のサムネイル一覧を表示する
- **THEN** 各映画サムネイルに映画詳細ページへのリンクを付与する

### Scenario: 存在しないカテゴリへのアクセス

- **WHEN** 存在しないカテゴリIDでアクセスされた
- **THEN** 適切なエラーメッセージまたは Not Found 表示をする

---

## Requirement: 映画詳細ページ

映画の詳細情報とコメント一覧を1画面で提供する。

**ユーザーストーリー**: 特定の映画に興味を持ったユーザーとして、その映画の詳細とほかのユーザーのコメントを確認したい。なぜなら視聴判断に必要な情報を一箇所で得られるからだ。

### Scenario: 映画詳細の表示

- **WHEN** ユーザーが映画詳細ページにアクセスする
- **THEN** 映画のタイトル・説明・like数をメインエリアに表示する
- **THEN** コメント一覧を右サイドバーに表示する

### Scenario: コメントの追加読み込み

- **WHEN** ユーザーがコメントをさらに読み込む操作をする
- **THEN** 続きのコメントを取得し、既存のコメント一覧に追記する

### Scenario: 存在しない映画へのアクセス

- **WHEN** 存在しない映画IDでアクセスされた
- **THEN** 適切なエラーメッセージまたは Not Found 表示をする

---

## 非機能要件

### パフォーマンス

- トップページは `homeScreens` クエリでカテゴリ＋映画一覧を一括取得する
  - API の `homeScreens` はエディターがキュレーションした動画リストとカテゴリが一体で返る設計のため、分割取得はできない
  - 初期表示の高速化は Next.js の ISR（`revalidate = 60`）でカバーする
- コメント一覧はページネーションにより必要な件数のみ取得する

### アクセシビリティ

- 映画サムネイル画像には代替テキストを付与する
- キーボード操作でページ間を移動できる

### レスポンシブ対応

- デスクトップ・モバイル双方で適切にレイアウトが調整される
